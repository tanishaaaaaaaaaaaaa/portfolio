## 👩‍💻 About

This is my personal developer portfolio — built from scratch without any frameworks or templates. It showcases my projects, skills, certifications, and achievements as a pre-final year **B.Tech student in Artificial Intelligence & Data Science**.

---

## ✨ Features

- **Fully Responsive** — works seamlessly on mobile, tablet, and desktop
- **Smooth Scroll Navigation** — fixed navbar with active scroll effect
- **Fade-in Animations** — scroll-triggered entrance animations using Intersection Observer
- **Downloadable Resume** — one-click resume download directly from the hero section
- **Working Contact Form** — opens the user's email client with pre-filled message
- **No Dependencies** — pure HTML, CSS, and vanilla JavaScript. Zero npm, zero frameworks
- **Cross-browser Compatible** — vendor-prefixed CSS for full Safari/Chrome/Firefox support

## 🌍 Deployment (Vercel)

This site is deployed on **Vercel** via GitHub integration.

Every push to the `main` branch automatically triggers a redeployment.

To deploy your own copy:

1. Fork this repository
2. Go to [vercel.com](https://vercel.com) and sign in with GitHub
3. Click **Add New → Project** and import this repo
4. Click **Deploy** — no configuration needed
